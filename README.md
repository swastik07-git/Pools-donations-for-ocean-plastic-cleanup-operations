# 🌊 Ocean Plastic Cleanup Donation Pool (Rust)

## 📌 Project Description
Ocean Plastic Cleanup Donation Pool is a lightweight Rust library for managing and tracking donations aimed at reducing ocean plastic pollution. It allows developers to create a simple system where contributions are pooled together, donors are recorded, and total funds are tracked transparently.

---

## ⚙️ What It Does
- Creates a donation pool  
- Accepts donations from users  
- Tracks total collected funds  
- Stores donor details for transparency  

---

## ✨ Features
- 💰 Donation pooling  
- 📊 Total donation tracking  
- 👥 Donor record management  
- ⚡ Fast and safe (Rust-based)  
- 🧪 Includes unit tests  

---

## 🛠️ Tech Stack
- Language: Rust  
- Type: Library (`lib.rs`)
